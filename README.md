# SurajMod Admin Panel

A **secure, advanced admin panel** for managing users, keys, themes, and access control. Built with **HTML, CSS, and JavaScript**, designed for simplicity and customization.

---

## **Features**

- User login system with **username & password**
- **Key system** with expiry for access control
- Manage multiple users: add, ban, remove
- Theme switcher: **default, cyber, matrix, galaxy, ocean, etc.**
- Dashboard with **status indicators** for keys and users
- Responsive UI with animations and effects:
  - Particles
  - Glitch
  - Waves
- Admin actions: reset keys, manage durations, ban users
